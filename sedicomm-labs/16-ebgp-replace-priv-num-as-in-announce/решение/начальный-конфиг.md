
# AS 500 routers

### R1
    ####preconfig
    interface lo0
    ip address 10.0.0.6 255.255.255.25

    ####ospf
    router ospf 1 
    network 10.0.0.0 0.0.0.3 area 0

    #####bgp
    router bgp 500
    #ebgp
    neighbor 10.0.20.1 remote-as 64512
    #ibgp
    neighbor 10.0.0.5 remote-as 500
    neighbor 10.0.0.5 update-source Loopback0
    neighbor 10.0.0.5 next-hop-self
